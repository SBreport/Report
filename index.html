API 키 무료 발급 방법

구글 계정(Gmail)만 있으면 무료로 발급 가능합니다.

STEP 1. 프로젝트 만들기

  1. Google Cloud Console 접속 → 구글 계정 로그인
  2. 처음이면 약관 동의 팝업 → 동의하고 진행
  3. 화면 상단 "프로젝트 선택" 클릭 → "새 프로젝트" 클릭
  4. 프로젝트 이름 아무거나 입력 (예: "유튜브보고서") → "만들기" 클릭

STEP 2. YouTube API 켜기

  1. YouTube Data API v3 페이지로 바로 이동
  2. 파란색 "사용" 버튼 클릭 → 끝!

STEP 3. API 키 만들기

  1. 사용자 인증 정보 페이지로 바로 이동
  2. 상단 "+ 사용자 인증 정보 만들기" 클릭 → "API 키" 선택
  3. API 키가 바로 생성됩니다 → 키를 복사해서 위 입력란에 붙여넣기

💡 완전 무료입니다. 하루 10,000유닛 제공되며, 월간 보고서용으로 충분합니다.
⚠️ API 키는 비밀번호처럼 관리하세요. 타인에게 노출되지 않도록 주의!
